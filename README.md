# Pill Reminder (web-based mobile app)

A small, mobile-friendly web app to track medicines and their schedules. Everything runs in the browser and is saved in `localStorage` (no backend needed).

## Features

- Add medicines with:
  - Name
  - Optional dose
  - Frequency:
    - Daily
    - Specific days of the week
    - Every other day
  - Time of day:
    - Morning, afternoon, night (any combination)
  - Start date and optional end date
- See all medicines in a simple list
- Remove individual medicines
- Clear all medicines

## Tech stack

- Plain **HTML**, **CSS**, and **JavaScript**
- Uses **localStorage** to persist data

## How to run

1. Open the `pill-reminder` folder.
2. Open `index.html` directly in a browser (double-click it or use "Open with").
3. For best results on mobile:
   - Open it in your mobile browser.
   - Add it to your home screen if your browser supports it.
